# Ashly Linktree Clone

A pixel-perfect clone of [Ashly's Linktree](https://linktr.ee/savingsguru.ca) as a custom landing page.

## Features

- Profile section with social links (Instagram, Facebook, TikTok, YouTube)
- Pill switcher for category filtering (All, Deals, Coupons, Apps, Shop, Downloads)
- Shop section with product cards
- Responsive Linktree-style design
- Smooth hover animations
- Exact color scheme matching (#d3d1c7 background, #39342c text)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
